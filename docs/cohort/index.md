# Cohort

This section holds the individual profiles and session documentation for each person in the coaching engagement.

Each coachee has their own space with a profile page (showing their current coaching phase, role, and assessment summary) and a session notes folder that grows over the course of the engagement.

Profiles are generated automatically from the coachee roster sheet. Session notes are synced from Google Drive after each session.
