# Practice

Structure without practice is theory. This phase is about using the new frameworks, boundaries, and approaches in real situations — and learning from what happens.

**Coaching posture:** Rehearsal partner; reality-tester.

**What this phase looks like:** Sessions become rehearsal spaces. We role-play conversations, anticipate friction, debrief real-world experiments, and refine approaches based on what actually works. Confidence grows through use, not instruction.

**Moving on:** When a person reports increased clarity and less emotional residue after taking action, they're ready to anchor what they've built.
